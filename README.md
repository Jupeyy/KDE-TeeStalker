# Installing

Just run the `build.sh` from the root of this project and add the new Widget (Tee stalker) to your KDE taskbar (usually the bottom control bar).

If you change your friend list, re-run the commmand and restart the plasma shell.

# What it does

Refreshes the master server list every minute and checks if one of your friend is online. If so the heart turns red. __You can click__ the heart to see a list of available friends:
